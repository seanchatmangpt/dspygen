# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2024.9.x | ✅ |

## Reporting a Vulnerability

Please report vulnerabilities via GitHub's private vulnerability reporting:
**GitHub → Security → Report a vulnerability**

Or email **info@chatmangpt.com** with subject `[SECURITY] dspygen`.

Include: description, reproduction steps, impact, and suggested fix if known.

We aim to respond within 72 hours and publish a fix within 14 days for critical issues.

## Scope

- Source code in `src/dspygen/`
- MCP server (`dspygen-mcp`)
- LSP server (`dspygen-lsp`)
- CLI (`dspygen`)

Out of scope: third-party LLM providers, user-supplied pipelines.

## Disclosure Policy

We follow coordinated disclosure. Please allow 90 days before public disclosure.
